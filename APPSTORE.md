## Description
KeySmith puts a powerful password generator right in your keyboard. No more switching apps, no more copy-paste juggling. Generate strong, unique passwords exactly where you need them.

**Generate Anywhere**
Switch to the KeySmith keyboard in any app and generate a password instantly. One tap to insert it directly into the text field.

**Your Password Vault**
Save generated passwords to your secure vault. Organized by categories, searchable, and protected by Face ID or Touch ID.

**Security You Can Trust**
- Cryptographically secure generation (SecRandomCopyBytes)
- Keychain-encrypted storage backed by the Secure Enclave
- Auto-clearing clipboard (30 seconds)
- Zero data collection. Zero tracking. Zero network access.
- Everything stays on your device. Period.

**Features**
- Custom iOS keyboard extension
- Four strength presets: PIN, Basic, Strong, Paranoid
- Passphrase mode (memorable word combinations)
- Adjustable length (4-64 characters)
- Customizable character sets
- Entropy strength meter (bits)
- Password vault with categories and search
- Face ID / Touch ID protection
- Dark mode support
- Full accessibility support (VoiceOver, Dynamic Type)

**Privacy**
KeySmith has no servers, no accounts, no analytics, and no telemetry. Your passwords never leave your device. We don't even know you exist.

## App Review Notes
This app includes a custom keyboard extension for generating passwords. The keyboard uses RequestsOpenAccess to enable clipboard functionality (copying generated passwords). No data is transmitted over the network. The keyboard extension shares password entries with the main app via Keychain Access Groups for a seamless user experience.

To test the keyboard: Settings > General > Keyboard > Keyboards > Add New Keyboard > KeySmith. Then tap the globe icon in any text field to switch to KeySmith.
